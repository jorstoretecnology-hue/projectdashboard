## 📜 Política de Archivado

### Qué se Archiva
- Handoffs de sesiones específicas (> 7 días)
- Reportes de seguridad históricos (> 3 meses)
- Documentación obsoleta o reemplazada
- Decisiones técnicas históricas

### Qué NO se Archiva
- Documentación activa (leer `../00-START-HERE.md`)
- Últimos 3 reportes de seguridad
- Handoff de la sesión actual

### Cuándo Archivar
- Al finalizar cada semana
- Al crear nueva versión de documentación
- Al completar una fase del roadmap
