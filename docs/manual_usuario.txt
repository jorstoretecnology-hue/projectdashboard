# 📟 MANUAL DE USUARIO: Portal de Seguimiento y Entrega Digital

¡Bienvenido! Este manual te guiará por el nuevo sistema de transparencia y seguridad para tus servicios y pedidos.

## 1. Inicio del Servicio o Pedido
Al contratar un servicio o realizar un pedido (reserva de glamping, orden de taller, pedido de restaurante, etc.), realizaremos un registro digital:
- Capturaremos los detalles y evidencia inicial (fotos, checklist de estado o preferencias).
- Recibirás un enlace único para seguir el progreso en tiempo real.

## 2. Seguimiento en Tiempo Real
Accede al enlace desde tu móvil para ver:
- El estado actual (Recibido, En Preparación, Listo para Entrega).
- Evidencia visual o detalles técnicos registrados al inicio.
- Resumen de ítems y presupuesto.

**Tip:** El portal es 100% responsivo. Puedes anclar la página a tu pantalla de inicio para un acceso más rápido.

## 3. Entrega y Firma Digital
Una vez que el servicio o producto esté listo para su entrega final:
1. **Identificación**: Ingresa tu número de documento para validar que el proceso es seguro y personalizado.
2. **Firma**: Firma directamente en la pantalla de tu móvil para confirmar la recepción a satisfacción.
3. **Cierre**: El sistema guardará legalmente tu conformidad y finalizará la gestión.

## 4. Gestión de Taller e Inspección 360° (Exclusivo Taller)
Para la industria de vehículos y equipos:
- **Inspección de Recepción**: Realizamos un escaneo de 7 puntos (luces, motor, estética, etc.) antes de iniciar.
- **Evidencia 360°**: Capturamos 4 fotos del estado actual para tu tranquilidad y la nuestra.
- **Transparencia**: Todo el checklist e imágenes estarán disponibles en tu portal de seguimiento.

## 🔒 Seguridad Avanzada
- **Contraseñas**: El sistema exige ahora contraseñas robustas (12+ caracteres, símbolos, números) para proteger tu cuenta empresarial.
- **Acceso Multi-Empresa**: Si gestionas varias sucursales, puedes alternar entre ellas desde el selector de "Control de Tenants" en un solo clic.
- **Protección**: Implementamos retardos de seguridad contra intentos de acceso no autorizados.
